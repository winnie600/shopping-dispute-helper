# C2C網購爭議協助幫手

## Description
這是一個能幫助解決C2C電商購物買家與賣家的退貨爭議問題的ai助手，尤其在C2C交易中往往退貨仰賴雙方的溝通，容易引發糾紛。這個AI助手能夠判別商品與描述不符的退貨分類,並判定買賣家責任,給出解決方法，實現AI智能判斷輔助和流程自動化。

## Target user
客服人員: 節省審查時間,提升準確度。  
買家 / 賣家:更快速且明確的回覆,降低爭議摩擦。  
平台管理者: 取得自動化報告。
